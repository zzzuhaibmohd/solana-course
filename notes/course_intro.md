# Solana

## What you will learn

- Solana program development
    - Native (Rust)
    - Anchor framework (Rust)

## Why should you take this course?

- Develop programs on Solana
- Practice Rust
- Audit and bug bounties

## Prerequisites

- Rust
- Blockchain basics
- Prior EVM smart contract development (Solidity, Vyper, etc...)

### Use of AI
Use of AI is encouraged for fast learning
1. Ask AI to write code or test
2. Fix code (using docs, Google search, AI, etc...)
